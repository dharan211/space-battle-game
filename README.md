🚀 Space Shooter Game

A simple 2D Space Shooter game built with Python and Pygame as an exploration project to learn game development fundamentals, understand the Pygame library, and experiment with core game mechanics such as movement, shooting, collision detection, and score tracking.

🎮 Features
Player-controlled spaceship
Enemy spawning and movement
Shooting mechanics
Collision detection
Score system
Sound effects and background music (if implemented)
Game Over screen
Smooth gameplay using Pygame's game loop
🛠️ Built With
Python 3.x
Pygame
